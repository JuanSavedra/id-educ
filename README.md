# ID.Educ

## Sobre o Projeto

O ID.Educ é uma plataforma de autogestão para alunos, projetada para auxiliar no acompanhamento do progresso acadêmico e desenvolvimento pessoal. A plataforma permite que os alunos registrem e visualizem suas notas, ocorrências e até mesmo identifiquem seu tipo de personalidade (MBTI).

## Status do Projeto

**Em andamento.**

Este projeto ainda está em desenvolvimento e não foi finalizado.

## Funcionalidades

*   **Gestão de Notas:** Adicione, visualize e acompanhe suas notas em diferentes disciplinas.
*   **Gestão de Ocorrências:** Registre e monitore ocorrências acadêmicas.
*   **Visualização de Dados:** Gráficos interativos para visualizar o desempenho acadêmico e a frequência de ocorrências.
*   **Perfil do Aluno:** Uma página de perfil para visualizar informações gerais do aluno.
*   **Teste de Personalidade:** Um seletor de MBTI para que os alunos possam identificar e registrar seu tipo de personalidade.

## Tecnologias Utilizadas

### Backend

*   **Java 17:** Linguagem de programação principal.
*   **Spring Boot:** Framework web para a criação da API REST.
*   **Maven:** Ferramenta de gerenciamento de dependências e build.

### Frontend

*   **React:** Biblioteca para a construção da interface de usuário.
*   **React Router:** Para gerenciamento de rotas na aplicação.
*   **Recharts:** Biblioteca para a criação de gráficos.
*   **TypeScript:** Superset do JavaScript que adiciona tipagem estática.
*   **Tailwind CSS:** Framework de CSS para estilização.
